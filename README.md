### What I learned

During this project, I improved my understanding of how to organize elements inside a component using Flexbox.

The main challenge was deciding how to properly position the elements inside the card and make them align correctly with the image border and the rest of the layout.

I practiced using Flexbox properties like `justify-content` and learned the importance of choosing the right value depending on the desired layout. In this project, I used `space-between` to distribute elements and `align-self` to adjust specific elements.

I also learned more about the BEM methodology. I decided to use it because it helped me keep my CSS organized and gave my component a clearer structure, making the card more modular and easier to maintain.
